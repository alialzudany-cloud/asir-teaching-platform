# منصة تجارب تحسين التدريس - تعليم عسير

منصة ويب محلية باللغة العربية بالكامل وباتجاه RTL لتوثيق وعرض وتحليل تجارب المدارس في تحسين التدريس، مع رفع ملفات حقيقي، لوحة إحصاءات، توليد ملخصات وقوالب تعريفية، وإصدارات قابلة للطباعة PDF.

## طريقة التشغيل

1. افتح الطرفية داخل مجلد المشروع:

```powershell
cd asir-teaching-platform
```

2. شغل المنصة:

```powershell
npm start
```

3. افتح الرابط:

```text
http://127.0.0.1:8020
```

## بيانات الدخول التجريبية

- مدير النظام: `admin@asir.local` / `Admin12345`
- مشرف الإدارة: `supervisor@asir.local` / `Supervisor12345`
- منسق مدرسة: `school@asir.local` / `School12345`

## ما الذي يعمل داخل النسخة

- تسجيل دخول بثلاث صلاحيات: مدير النظام، مشرف الإدارة، مدرسة / منسق مدرسة.
- نموذج تسجيل تجربة على ست خطوات مع حفظ كمسودة وإرسال للمراجعة.
- رفع ملفات حقيقي داخل مجلد `uploads` مع التحقق من الامتدادات والحجم.
- دعم ملفات PDF والصور وMP4 وExcel CSV/XLSX وWord DOCX وPowerPoint PPTX.
- صفحة استعراض التجارب مع بحث وفلاتر حسب الحالة والمجال.
- صفحة تفاصيل التجربة مع المرفقات والإجراءات الإدارية والحذف النهائي لمدير النظام.
- اعتماد التجربة أو إرجاعها للتعديل أو رفضها.
- توليد ملخص احترافي وبطاقة تعريفية بقالب عربي داخلي.
- لوحة إحصاءات محسوبة من البيانات المدخلة.
- تصدير كل التجارب بصيغة CSV متوافقة مع Excel.
- صفحات طباعة جاهزة للتصدير PDF للتجربة الواحدة، الإحصاءات، والإصدارات.
- إصدار من التجارب المعتمدة، مع فهرس وصفحات تجارب ورموز QR شكلية قابلة للاستبدال.

## قاعدة البيانات والملفات

- يتم إنشاء قاعدة بيانات أولية تلقائيًا في:

```text
data/db.json
```

- تحفظ الملفات المرفوعة في:

```text
uploads/
```

تم اختيار تخزين JSON محلي بلا حزم خارجية حتى تعمل النسخة فورًا بأمر واحد في بيئة لا تسمح بتنزيل الاعتماديات. يمكن لاحقًا نقل نفس الجداول إلى SQLite أو PostgreSQL لأن أسماء الحقول مطابقة تقريبًا للهيكل المطلوب.

## نقاط التوسعة المستقبلية

- ربط OpenAI API داخل دوال توليد الملخص والبطاقة الموجودة في `server.js`.
- استبدال التخزين المحلي بـ SQLite أو PostgreSQL.
- استبدال الطباعة من المتصفح بـ Puppeteer عند توفر الاعتماديات.
- إضافة تحكيم ودرجات ومقارنات بين المدارس.
